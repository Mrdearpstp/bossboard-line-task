วิธีเริ่มใช้งาน

  • เพิ่มเพื่อน LINE Official Account ของ BossBoard
  • เปิดแอปผ่าน LINE MINI App หรือปุ่มเปิดแอปในแชท
  • อนุญาตให้แอปเข้าถึงโปรไฟล์ LINE เพื่อแยกข้อมูลงานของคุณ
  • พิมพ์งานในแชท เช่น “ประชุมพรุ่งนี้ 10 โมง” หรือ “ส่งรายงานวันที่ 31”

ปัญหาที่พบบ่อย

  • เปิดแอปแล้วขึ้น 404: ตรวจสอบว่าใช้ Published LIFF URL หรือแอปผ่านการ publish แล้ว
  • บอทไม่ตอบ: ตรวจสอบว่า Messaging API webhook เปิดใช้งาน และ Webhook URL ถูกต้อง
  • ไม่ได้รับแจ้งเตือน: ตรวจสอบการตั้งค่าแจ้งเตือนในหน้า LINE ของแอป และตรวจสอบว่าเพิ่มเพื่อน OA แล้ว
  • ข้อมูลไม่ตรงผู้ใช้: เปิดแอปผ่าน LINE และกดยืนยัน/เชื่อม LINE อีกครั้ง

  • ข้อกำหนดการใช้งาน
