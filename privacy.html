นโยบายความเป็นส่วนตัว

อัปเดตล่าสุด: 29 พฤษภาคม 2026

BossBoard เป็นแอปช่วยจดงานและแจ้งเตือนผ่าน LINE เพื่อให้ผู้ใช้บันทึกสิ่งที่ต้องทำ จัดกลุ่มตามโปรเจกต์ และรับการแจ้งเตือนงานของตนเองได้ง่ายขึ้น นโยบายนี้อธิบายว่าเราเก็บ ใช้ และดูแลข้อมูลของคุณอย่างไร

1. ข้อมูลที่เราเก็บ

  • ข้อมูลโปรไฟล์พื้นฐานจาก LINE เช่น LINE user ID, ชื่อโปรไฟล์ และรูปโปรไฟล์ เมื่อคุณอนุญาตให้แอปเข้าถึง
  • ข้อมูลงานที่คุณสร้าง เช่น ชื่องาน รายละเอียด โปรเจกต์ วันครบกำหนด เวลาเตือน สถานะ และความสำคัญ
  • การตั้งค่าการแจ้งเตือน เช่น เวลาแจ้งเตือน สรุปรายวัน การเตือนก่อนครบกำหนด และช่วงเวลางดแจ้งเตือน
  • บันทึกกิจกรรมในงาน เช่น การสร้างงาน แก้ไขงาน เปลี่ยนสถานะ หรือเลื่อนกำหนด

2. เราใช้ข้อมูลเพื่ออะไร

  • เพื่อแสดงรายการงานและโปรเจกต์ของผู้ใช้แต่ละคน
  • เพื่อสร้างงานจากข้อความที่ผู้ใช้พิมพ์ใน LINE
  • เพื่อส่งการแจ้งเตือน สรุปงาน หรือสถานะงานกลับไปยัง LINE ของผู้ใช้
  • เพื่อปรับปรุงประสบการณ์ใช้งาน เช่น การตั้งค่าเริ่มต้นและการแสดงผลในแอป

3. การแยกข้อมูลของผู้ใช้

ข้อมูลใน BossBoard ถูกผูกกับ LINE user ID ของแต่ละผู้ใช้ เพื่อให้ผู้ใช้เห็นเฉพาะข้อมูลงานของตนเอง เว้นแต่ในอนาคตผู้ใช้เลือกเข้าร่วมฟีเจอร์ทีม/องค์กรที่มีการกำหนดสิทธิ์ร่วมกัน

4. การแบ่งปันข้อมูล

เราไม่ขายข้อมูลส่วนบุคคลของผู้ใช้ให้บุคคลภายนอก ข้อมูลจะถูกใช้เฉพาะเพื่อให้บริการ BossBoard และการเชื่อมต่อกับ LINE เท่านั้น ในกรณีที่ต้องใช้ผู้ให้บริการโฮสติ้งหรือฐานข้อมูล ข้อมูลอาจถูกประมวลผลผ่านระบบเหล่านั้นตามความจำเป็นในการให้บริการ

5. การเก็บรักษาและลบข้อมูล

เราเก็บข้อมูลเท่าที่จำเป็นสำหรับการให้บริการ ผู้ใช้สามารถลบงาน ล้างงานที่เสร็จแล้ว หรือขอลบข้อมูลทั้งหมดของตนเองได้ ผ่านเมนูในแอปหรือโดยติดต่อผู้ดูแลบริการ

6. ความปลอดภัย

เราใช้การเชื่อมต่อ HTTPS และจำกัดการเข้าถึงข้อมูลตามผู้ใช้ LINE เพื่อช่วยป้องกันการเข้าถึงข้อมูลโดยไม่ได้รับอนุญาต อย่างไรก็ตาม ไม่มีระบบออนไลน์ใดที่ปลอดภัยสมบูรณ์ 100%

7. บริการของ LINE

การเข้าสู่ระบบหรือใช้งานผ่าน LINE อยู่ภายใต้นโยบายและข้อกำหนดของ LINE ด้วย ผู้ใช้ควรอ่านนโยบายของ LINE เพิ่มเติมหากต้องการทราบรายละเอียด
